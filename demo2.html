Popup

Demo 2 on jatkoa aiemmin tekemälleni Demo 1-sivustolle. Tällä kertaa tein uuden aloitussivun, jonne sijoitin linkit Demo 1- ja Demo 2 -sivuille. Lisäksi tein demojen sivuille navigointilinkin takaisin etusivulle.

Toteutus

Toteutuksessa hyödynsin w3schools.com -sivustoa sekä Demo 1:n html- ja css-tiedostoja. Aloitin luomalla uuden html-tiedoston nimellä Demo 2 sekä uuden css-tiedoston. Sen jälkeen lisäsin html-tiedostoon viittauksen css-tiedostoon. Aloitussivua varten tein uuden index.html -tiedoston. Tämän jälkeen loin JavaScript-tiedoston. Demo 2 sivun popup-ikkunan toiminnallisuus on luotu JavaScriptillä. Muutoksia tuli matkan varrella useita. Kaikki muutokset vein lokaaliin repositoryyn Git-versionhallintaan.

Tallensin ja testasin koodin toimivuutta selaimella säännöllisesti. Muutosten jälkeen tein uuden git commitin. Lopuksi lisäsin GitHub -repositoryn versionhallinnan lähteeksi ja pushasin lokaalit muutokset lähderepositoryyn.
